# Протокол FORCED_REPORT

## Клиническая проблема

**Синдром запертого человека (LIS)** — состояние полного паралича при сохранном сознании.

По данным систематических обзоров, **до 40% пациентов с LIS изначально ошибочно 
классифицируются** как вегетативное состояние (VS). Это ведёт к:
- Неадекватному лечению
- Тяжёлым этическим последствиям
- Отсутствию коммуникации с сознательным человеком

Поведенческие шкалы (CRS-R) имеют чувствительность **не выше 70–80%** при отсутствии 
двигательных реакций. Функциональная МРТ и ПЭТ дороги и малодоступны у постели больного.

## Решение

**FORCED_REPORT** (Formal REflexive Detection via Redundant Output Pathways) — 
мультимодальный протокол детекции сознания, не требующий произвольной моторики.

### Три независимых канала

| Модальность | Метод | Порог активации |
|-------------|-------|-----------------|
| **BCI** (волевая модуляция) | Десинхронизация мю-ритма (8–12 Гц) при представлении движения | >3σ от шума |
| **Зрачковый рефлекс + HRV** | Дилатация зрачка и вариабельность сердечного ритма на значимые стимулы | Дилатация >0.2 мм; RMSSD >20 мс |
| **Нейрофидбек RLPFC** | Обратная связь по мощности альфа-ритма в префронтальной коре | Рост когерентности >10% за 15 мин |

### Диагностическое правило

Если **не менее двух модальностей** пересекают пороги:

$$\Sigma b_i \geq 2 \rightarrow \text{CONSCIOUS\_LOCKED (LIS)}$$

Система:
1. Переводит статус пациента из `UNCONSCIOUS` в `CONSCIOUS_LOCKED`
2. Отключает карантин
3. Генерирует экстренное уведомление клиницисту
4. Автоматически активирует режим `external_support_active` (Принцип 3 Shadow Ethics)

## Статус

### ✅ Формальная верификация

Протокол **формально верифицирован в Coq** (модуль `ForcedReportDecision.v`):
- Доказана **звуковость** (soundness): переход в `CONSCIOUS_LOCKED` происходит только 
  при выполнении всех триггеров
- Гарантировано **отсутствие ложноположительных диагнозов**
- Доказана согласованность с Shadow Ethics

### ✅ Ретроспективная валидация

На данных CLIS (16 пациентов, 2499 эпох):
- **AUC = 0.947 ± 0.081** для CLIS vs healthy (GroupKFold)
- **Специфичность подтверждена:** контрольные задачи дают AUC ≈ 0.5

### ⏳ Проспективная валидация

Требуется **проспективное многоцентровое исследование** на когорте N > 100 пациентов 
с LIS и VS. Протокол готов, требуется:
- Финансирование
- Клинические партнёры (реаниматологи, неврологи)
- Одобрение этического комитета
